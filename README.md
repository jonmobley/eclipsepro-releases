# EclipsePro

EclipsePro is a macOS presentation and display-control app for live events: drive external displays with images, videos, slideshows, web pages, PDFs, cameras, and screen shares — with freeze, blackout, a presenter pointer, a web remote, and EclipseCast live streaming to browsers on your network.

## Download

**[Download the latest version](https://github.com/jonmobley/eclipsepro-releases/releases/latest/download/EclipsePro-Installer.dmg)** (macOS 15.6 or later, Apple silicon and Intel)

Open the DMG and drag Eclipse to Applications. The app is notarized by Apple and updates itself automatically.
